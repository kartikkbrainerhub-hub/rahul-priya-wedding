# Rahul ❤️ Priya - Wedding Invitation

A premium, mobile-first, and highly cinematic Indian wedding invitation website built with Next.js, Framer Motion, and Tailwind CSS. Designed to deliver a luxurious, immersive guest experience with custom particle backgrounds, floating petals, romantic ambient music, and a premium masonry photo gallery.

## ✨ Features

- 🌟 **Cinematic Theme**: Luxurious dark romantic color palette (Royal Gold, Deep Maroon, Crimson, and Cream) inspired by Indian wedding aesthetics.
- 🎬 **Animated Backgrounds**: Falling flower petals and gentle glowing candle particle animations (fully GPU-accelerated).
- 🎵 **Romantic Soundscape**: Integrated background music player with elegant, smooth audio controls.
- 📸 **Premium Photo Gallery**: An interactive masonry gallery with lightboxes containing stunning high-fidelity images of the royal ceremonies.
- 🗓️ **Interactive Timeline & Map**: Visually beautifully arranged wedding ceremonies, timing, and venues with calendar additions.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kartikkbrainerhub-hub/rahul-priya-wedding.git
   cd rahul-priya-wedding
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000).
